# Neutral Procurement Prototype

This is a standalone clickable mockup for an Intelligent Procurement Automation Platform.

The prototype intentionally uses neutral branding and mock data only. It does not reference real client, company, ERP, or implementation partner names.

## Pages Included

- Dashboard
- PR Review Workbench
- Query Support
- Vendor Review & Decision Pack
- Reports placeholder

## Hosting

This folder is deployed as a static site and can be served from GitHub Pages, Netlify, Replit, or similar.
